Enable number input from the keyboard in addtion to the buttons.


The notes feature can be toggled on and off with the space bar in addition to clicking the button.


Highlight adjacent boxes using the arrow keys in addition to clicking the box.


Quickly navigate to a box using just the numperpad. When nothing is highlighted, pressing a number on the numpad will highlight the respective square, and pressing another number will highlight the respective box in that square. For example, pressing 7 with nothing highlighted will highlight the top left square, since 7 is in the top left of the numpad. Pressing 2 will then select the bottom middle box in the top left square. Pressing 5 will input 5 into the highlighted box and then remove the highlight from that box. Pressing Escape will deselect any box or square.


Values in the wrong spot will be displayed in red


If a value is placed in a box, all notes with that value will be removed if that value can no longer go there.
